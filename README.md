# SOAT Manager Hero

Sistema de gestión de pólizas SOAT para Holding Group - Hero

## Descripción
Sistema web para administrar la bolsa prepago de pólizas SOAT, permitiendo el control de recargas, expedición de pólizas y seguimiento del saldo.

## Cliente
Holding Group - Hero

## Tarifas 2026
- Moto hasta 99cc: $256,200 + Comisión $30,000 = $286,200
- Moto 100-200cc: $343,300 + Comisión $30,000 = $373,300

## Tecnologías
- **Backend**: FastAPI + PostgreSQL
- **Frontend**: React + Vite + TypeScript + Tailwind CSS
- **Deployment**: VPS Hostinger

## Estructura del Proyecto
```
soat-manager-hero/
├── backend/          # API FastAPI
├── frontend/         # React App
└── README.md
```

## Roles
- **Administrador**: Control total del sistema
- **Cliente**: Consulta de saldo y SOATs expedidos
